### OpenCode Server Pool (Performance)

OMCM uses a **flexible server pool** to reduce **routing call latency by ~90%** compared to CLI mode:

```bash

# Manual server control is still available:
```

| Mode | First Call | Subsequent Calls |
|------|------------|------------------|
